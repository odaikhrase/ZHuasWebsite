# Decor Emperor

Premium Marble PVC and WPVC Wall Panels in Columbus, Ohio.

## Project Description
Decor Emperor specialize in luxury marble PVC wall panels and WPVC decorative sheets. Our products are durable, waterproof, and affordable, making them ideal for both residential and commercial spaces.

## Tech Stack
- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## Development
```sh
npm install
npm run dev
```

## Build
```sh
npm run build
```
